# Библиотека иконок

Личная коллекция SVG-иконок для будущего использования в RideTrack (пока не подключена к коду — это только место для накопления файлов).

## Структура

- `fluent/` — цветные иконки (Fluent).
- `material/` — вторая коллекция (рабочее название «Material» — переименуйте папку, если найдётся более точное имя набора).

Обе — плоские папки, без вложенных категорий: поиск идёт по имени файла, вложенность только мешала бы.

## Правило именования файлов (обязательно для поиска по ключевым словам)

Имя файла = ключевые слова через дефис, на английском, в нижнем регистре, без цифр-версий и служебных префиксов:

```
battery-charging.svg
car-front.svg
credit-card.svg
map-pin.svg
```

Чем точнее и типичнее слово (как в обычном поиске), тем легче потом найти иконку через `grep`/поиск по файлам — никакой отдельной базы данных или JSON-манифеста не заводим, файловая система и есть индекс.

Если у иконки несколько смысловых синонимов — перечисляйте через дефис все, а не выбирайте один:

```
car-auto-vehicle.svg
```

## Формат файла

Только `.svg`. Для `fluent/` (цветные) — оставляйте встроенные цвета как есть. Для `material/` — как есть в исходном наборе (обычно однотонные, `currentColor` или чёрный).
